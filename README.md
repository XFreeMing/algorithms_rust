# 算法+RUST学习
## 安装Rust
1. 安装Rust
    ```bash
    curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
    ```
2. 配置环境变量
    ```bash
    source $HOME/.cargo/env
    ```
3. 验证安装
    ```bash
    rustc --version
    ```
4. 升级Rust
    ```bash
    rustup update
    ```
5. 卸载Rust
    ```bash
    rustup self uninstall
    ```
